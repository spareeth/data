Deploying

  1. Set the OPENAI_API_KEY environment variable.
  2. Optional: set SECRET_KEY for session security.
  3. Install dependencies: pip install -r requirements.txt.
  4. Start the server: flask --app app run --host 0.0.0.0 --port 5000.
